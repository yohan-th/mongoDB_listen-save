Le simulateur lit un fichier decrivant l'enchainement des actions:
# simulateur < simu
options
-v imprime les actions au fur et a mesure
-s imprime les message JSON sur la sortie standard
le fichier simu est commente est peut facilement etre modifie, en particulier pour ajuster les temps

Pour le debug, utilisr la commande lecture qui afficher les messages JSON envoyes
